Cliente desktop pronto

Este aplicativo abre URLs externas em uma janela local controlada quando acionado pela aplicacao web.

Agente local:

http://127.0.0.1:43125/health

Formato do deep link:

urlembeddiag://open?url=https%3A%2F%2Fexample.com

Apenas URLs http:// e https:// sao aceitas. O aplicativo nao tenta contornar login, CAPTCHA, Cloudflare ou bloqueios de autorizacao do site de origem.
